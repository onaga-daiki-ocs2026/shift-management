# Shift Management App（シフト管理アプリ）

## 概要

アルバイト先などで行われるシフト提出・管理作業を効率化するためのWebアプリケーションです。

従来の紙やメッセージアプリでのシフト提出では、
管理者側の集計作業や確認作業に時間がかかるという課題があります。

本アプリでは、スタッフがスマートフォンから希望シフトを提出し、
管理者が提出状況を確認・管理できる仕組みを提供します。


## 開発背景

実際のアルバイト現場で、
シフト希望の回収・確認作業が手間になっていることをきっかけに開発しました。

単なる入力フォームではなく、

- スタッフ側の提出しやすさ
- 管理者側の確認しやすさ
- スマートフォン利用を前提とした操作性

を意識して設計しています。

## 主な機能

### スタッフ機能

### シフト希望提出

- 14日単位でシフト希望を入力
- 出勤可能日・時間を登録
- 複数期間分の提出に対応予定

### 提出済みシフト確認

- 自分が提出した希望シフトを一覧表示
- 日付・時間ごとに確認可能


## 管理者機能

### 希望シフト一覧確認

- スタッフから提出されたシフト希望を一覧表示
- 管理者用画面から確認可能


### 確定シフト作成（開発予定）

- 提出された希望シフトをもとに勤務表を作成
- 視覚的に調整できるUIを検討中

## データ設計（一部）

### User

| Column | 内容 |
| --- | --- |
| id | ユーザーID |
| lineUserId | LINEユーザーID |
| name | 名前 |
| role | 権限 |


### ShiftRequest

| Column | 内容 |
| --- | --- |
| id | シフト希望ID |
| user | 提出者 |
| workDate | 希望日 |
| startTime | 開始時間 |
| endTime | 終了時間 |
| available | 出勤可否 |
